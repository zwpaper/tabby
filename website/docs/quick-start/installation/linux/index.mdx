# Linux
Running Tabby on Linux using Tabby's standalone executable distribution.

## Download the Linux release
1. Go to the Tabby release page: https://github.com/TabbyML/tabby/releases.
2. Click on the **Assets** dropdown for a specific release to find the manylinux zip files.

   {/* FIXME(wei): update the download screenshot for linux */}
   <div align="left">
     <img src={releaseImage} alt="Linux release" width="626" style={{ borderRadius: "12px" }} />
   </div>

3. Click to download the release **tabby_x86_64-manylinux2014.zip**.
4. Untar the file you downloaded to get the `tabby` executable.

   ```bash
   tar -zxvf tabby_x86_64-manylinux2014.zip
   ```

5. Move `tabby` to a folder under `$PATH` or other one of your choice.

## Running Tabby

Run the following command:

```bash
# If tabby is under $PATH
tabby serve

# If tabby is not under $PATH
cd path/to/tabby
./tabby serve
```

You should see a success message similar to the one in the screenshot below.
After that, you can visit [http://localhost:8080](http://localhost:8080) to access your Tabby instance.
