# Windows
Running Tabby on Windows using Tabby's exe distribution.

## Download the Windows release

1. Go to the Tabby release page: https://github.com/TabbyML/tabby/releases
2. Click on the **Assets** dropdown for a specific release to find the Windows zip files.

   {/* FIXME(wei) update the windows download screenshot */}
   <div align="left">
     <img src={releaseImage} alt="Windows release" width="800" style={{ borderRadius: "12px" }}/>
   </div>

3. Download the release, **tabby_x86_64-windows-msvc.zip**.
4. Unzip the file `tabby_x86_64-windows-msvc.zip`.
5. Find the Windows executable file `tabby.exe`.

## Running Tabby

Open a command prompt or PowerShell window, as administrator, in the directory where the `tabby.exe` is located (from the previous step).

Run the following command:
```powershell
.\tabby.exe serve
```

You should see a success message similar to the one in the screenshot below.
After that, you can visit [http://localhost:8080](http://localhost:8080) to access your Tabby instance.
