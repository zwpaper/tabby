# Docker Compose

This guide explains how to launch Tabby using docker-compose.

1. First create the `docker-compose.yml` file in your project directory:

    ```yaml title="docker-compose.yml"
    version: '3.5'

    services:
      tabby:
        restart: always
        image: registry.tabbyml.com/tabbyml/tabby
        entrypoint: /opt/tabby/bin/tabby
        command: serve
        volumes:
          - "$HOME/.tabby:/data"
        ports:
          - 8080:8080
    ```

2. Run the following command to launch Tabby:

    ```bash
    docker compose up -d
    ```

After Tabby is running, you can access it at [http://localhost:8080](http://localhost:8080).

To view the logs, you can use the following command:

```bash
docker compose logs
```
